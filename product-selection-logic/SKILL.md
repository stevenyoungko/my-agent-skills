## 建立好你的心態

不是你喜歡什麼就賣什麼，而是要根據市場上的需求缺口進行選品，

在需求大於供給的領域中選擇，那你就可以很輕鬆的獲得客源。

舉例來說：

我的開久選物就是觀察到市場上寬大且品質穩定的服飾太少所以決定在這個領域發展。

## 商品壽命

未來趨勢是決定這個商品是否可以持續賣個五年甚至十年，

例如：穿過寬褲的人很難再穿回合身的褲款因為風格相差甚遠，

就代表他未來會持續穿寬褲且購買

或是說人口高齡化那未來可能老人用品的需求會持續增長 該商品的壽命就會拉長

## 自身熱忱

雖然不能任性的選擇賣自己喜歡的，但也不能賣一個你完全沒有興趣的商品

販售任何的商品在初期可能都有新鮮感，但做了一兩年後就會對於制式化的上架疲乏職業倦怠，所以盡量還是挑選會賺錢同時你比較感興趣的商品切入

## 輔助工具

Google Trends 可以幫你查看搜尋趨勢，找點靈感，另外也可以透過Google Trends來簡單驗證自己想出的選品邏輯是否正確

小紅書或是Tiktok，相較國內常用的社群媒體，國外的社群媒體會有更多信息差，在國外可能已經賣很火的產品，國內還沒有人經營

## 眼光培養

大量閱讀瀏覽、IG小紅書追蹤、Pinterest、Youtube、各大電商平台多逛、實體店家細品、時間積累、觀察世界、接觸人群、以老闆角度看同行、以顧客角度看市場 ( 找缺口 )

## 選品範例

1. 閱讀趨勢、延伸思考

例：透過財經頻道提出疫後報復性旅遊 ⇒ 行李箱

2. 實地尋找、留意特色

例：中國出差認真尋找商機 ⇒ 水果咖啡

3. 記下可改善的事情

例：喝酒沒新花樣玩 ⇒ 喝酒道具、小物

4. 輔助工具

例：小紅書發現Y2K市場、Tiktok發現火紅玩具

5. 接觸人群、感受變化

例：好多朋友在露營、一逛發現裝備好貴 ⇒ 白牌露營用品

6. 體驗市場、發現缺口

例：沒辦法在蝦皮上買到喜歡的皮帶 ⇒ 配件店

例：送禮物超難想 ⇒ 禮物店+好笑產品

7. 自身喜好去發掘 ( 小心驗證市場 )

例：喜歡貓狗 ⇒ 開設寵物服飾、周邊店

## 流程

1. 建立觀念
2. 思考

## 選品評分表

### 核心評分

| 項目 | 評分範圍 | 淘汰門檻 |
|------|----------|----------|
| 需求大於供給 | 1 - 10 分 | 低於 5 分直接淘汰 |
| 產品壽命 | 1 - 5 分 | 低於 3 分直接淘汰 |
| 自身熱忱 | 1 - 5 分 | 低於 2 分直接淘汰 |
| 毛利高低 | 1 - 9 分 | 低於 3.5 分直接淘汰 |
| 市場規模 | 1 - 5 分 | 低於 1.5 分直接淘汰 |

### 額外加分

| 加分項目 | 分數 |
|----------|------|
| 明確痛點 | +1 |
| 低耗損 | +1 |
| 垂直性 | +1 |
| 話題性 | +1 |
| 產品體積小 | +1 |
| 低上架頻率 | +1 |
| 工作流程簡易 | +1 |
| 有差異化、故事性 | +1 |
| 中黏著性 | +1 |
| 高黏著性 | +2 |
| 照片好呈現 | +2 |
| 中回購屬性 | +2 |
| 高回購屬性 | +3 |

### 計算公式

**核心評分 × 2.5 + 額外加分 = 選品評分**

- 滿分：100 分
- 及格分：44 分以上
